
# 💰 AI Personal Finance Advisor 

An AI-powered personal finance assistant that helps users manage budgets, track expenses, analyze financial health, and receive intelligent financial advice using CrewAI and Gemini.

---

## 🚀 Features

- 💸 Expense Tracking (Rent, Food, Transport, etc.)
- 📊 Financial Summary Dashboard (Income, Expenses, Savings)
- 📉 Expense Breakdown (Pie Chart - safe & error-free)
- 📊 Financial Health Visualization (Bar Chart)
- 🧠 AI Financial Advisor (CrewAI + Gemini)
- 🎯 Savings Goal Tracker
- 💾 Persistent Data Storage (JSON)
- 📥 Downloadable Financial Report

---

## 🛠️ Tech Stack

- Python
- Streamlit
- CrewAI (Multi-Agent AI)
- Google Gemini API
- Matplotlib
- NumPy

---

## ⚙️ Setup Instructions

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/ai-finance-advisor.git
cd ai-finance-advisor
````

---

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 3️⃣ Setup Environment Variables

Create a `.env` file:

```env
GOOGLE_API_KEY=your_gemini_api_key
```

Get your API key here:
👉 [https://makersuite.google.com/app/apikey](https://makersuite.google.com/app/apikey)

---

### 4️⃣ Run the App

```bash
streamlit run app.py
```

---

## 🧪 How to Use

1. Enter your monthly income
2. Add expenses (rent, food, transport, etc.)
3. Set a savings goal
4. View:

   * Financial summary 📊
   * Expense distribution 📉
   * Savings progress 🎯
5. Click **Get Advice** for AI insights
6. Download your report 📥

---

## 📊 Output Includes

* Total income, expenses, savings
* Expense category breakdown
* Financial health overview
* AI-generated financial advice
* Savings goal progress
